Meine momentane Idee ist ein einfachegehaltenes Webspiel,dass eine Mischung aus Quiz-/Rhythmus-
und Reaktionsspiel werden soll, zu programmieren.

Der User bekommt per Zufall bestimmte Aufgaben, z.b. Quizfragen, Rythmusaufgaben(drücke die Knöpfe in der Reihenfolge in der sie leuchten) oder Aufgaben wie "Drücke den roten Knopf in 5 Sekunden".
Während er diese bearbeitet misst ein Timer die Zeit, die der User im Spiel verbringt. 
Die "Überlebenszeit" wird nach dem verlieren des Users angezeigt und ist der "Score" des Spieles, woran der User sehen kann,
ob er sich gut geschlagen hat.

Da dieses Spielprinzip auf Dauer langweilig werden könnte, werden Extras eingebaut.
Z.B.soll der User ein Hintergrundbild, sowie eine Hintergrundmusik wählen dürfen.

Abgesehen davon soll das Programm später erweitert werden.
Damit meine ich, dass man leicht neue Aufgabenstellungen hinzufügen kann, beispielsweise, dass der User 
Antworten schreiben oder Lückentexte füllen soll. 
Auf diese Weise sollen die Interaktionsmöglichkeiten auch nach Belieben erhöht werden.




Wäre so eine Anwendung möglich für Projekt 1?

=====Projektidee 8.5.2016===================================
Meine Projektidee ist die Umsetzung des Brettspiels "Civilisation".

=======Kurze Zusammenfassung=======
"Civilisation" ist ein strategisches Brettspiel, indem der Spieler ein Volk wählt (eine Zivilisation) 
und mit dem Volk Städte errichtet. In den Randgebieten der Städte erwirtschaftet er Rohstoffe, welche er 
nutzen kann um Gebäude zu errichten und Einheiten/Figuren zu generieren. 

Das Spiel kann mit 2 bis 4 Spielern gespielt werden. 
Alle Spieler konkurieren gegeneinander um die Fortschritte in ihren Zivilisationen. 
Mit dem Begriff Fortschritt sind die Bereiche "Technologie/Forschung", "Kultur", "Wirtschaft" und "Millitär" gemeint.
Wenn ein Spieler einen dieser Bereiche quasi komplett ausgebaut hat, hat er gewonnen.

Quasi schreibe ich, weil in einige Bereiche Ausnahmen sind.
Z.b. gilt der Sieg durch Millitär erst, wenn man die Hauptstadt eines anderen Spielers erobert hat mit dem Millitär.

Genaueres steht in der ausführlichen Zusammenfassung.

					-Kommunizieren mit anderen Spielern [Markt; Chat]
					-(strategisch die nächsten Ereignisse planen [Wie auf welche Art man siegen möchte; was man tun wird])








=======Ausführliche Zusammenfassung====

==Vor Spielbeginn==
Zum Spielbeginn einigen sich die Spieler entweder auf die eigenen Völker oder lassen es per Zufall bestimmen.
Danach bekommt jeder einen Zivilisationsbogen, einen Kulturstufenmarker und einen Heimatspielplanteil.

Jedes dieser Völker besitzt eine eigene individuelle Kultur, welche mittels des Zivilisationsbogens und dem Kulturstufenmarker dargestellt wird.
Die Kulturstufen lassen sich im Spielverlauf erhöhen und bringen dem Spieler Vorteile. [Dazu später mehr]
Der Heimatspielplanteil bestimmt den Kartenabschnitt, wo der jeweilige Spieler das Spiel beginnt.

==Spielablauf==
Während jeder Runde besitzen die Spieler mehrere mögliche Interaktionsmöglichkeiten, die als einzelne Phasen gegliedert sind.
Sie können:
1. "Rundenbeginn": Neue Städte gründen (wenn das Spielfeld es erlaubt(nicht erlaubt auf z.B. Wasser))
2. "Handelsphase": Handeln mit anderen Spielern (wenn sie Handelspunkte besitzen)
3.  "Städteverwaltung": Eine Stadtaktion pro Stadt ausführen (ENDWEDER: Eine Figur,eine Einheit, ein Gebäude,ein Weltwunder produzieren  oder  einen Rohstoff abbauen  oder  sich den Künsten widmen)
4. "Bewegungsphase"
5.  Forschen



1. ==Rundenbeginn - Städte gründen == 
	Zu Beginn darf der Spieler eine neue Stadt gründen. 
	Dafür muss der Spieler mit einer Pionierfigur auf dem Feld sein, wo seine Stadt entstehen soll. Im Austausch für die neue Stadt verschwindet die Pionierfigur.
	Dies geht aber nur wenn folgende Bedingungen auf das Feld zutreffen:
			- Auf einem Feld liegen, die kein Wasserfeld ist.
			- Muss um 8 offene umliegende Felder besitzen (Also z.b. nicht am Spielfeldrand).
			- Darf nicht direkt an Hütten oder Barbarendörfern anliegen.
			- Das Feld darf nicht an eine gegnerische Figur angrenzen.
			- Das Feld muss mindestens 3 Felder Abstand zu jeder anderen Stadt haben.

Das Mittelfeld wird das Stadtzentrum bilden und die 8 umliegenden Felder die Randbezirke.
Die Randbezirke sind in der Lage Rohstoffe zu produzieren (3.).
Andere Spielfiguren des Spielers, die auf dem selben Feld standen werden auf umliegende Felder verschoben, wen die Stadt entsteht.
Die Stadt wird mit einem Stadtmarker belegt und so markiert.
Die Stadt darf direkt in der ersten Runde schon Handeln.

2. ==Handelsphase==
	In der handelsphase dürfen die Spieler handeln und Handelspunkte erhalten.
	In der Handelsphase erhält der Spieler für jedes Handelssymbol in all seinen Städten Handelspunkte.
	In den Zivilisationsbogen befindet sich eine Handelsscheibe, deren Anzeige nun um den Betrag der Handelspunkte erhöht wird.
	Die Handelspunkte können nun für den Handel verbraucht werden. Falls nicht, bleiben diese Vorhanden und können in den nächsten Runden genutzt werden.
	In der nächsten Runde werden erneut neue Handelspunkte hinzugerechnet.
	Ein Spieler kann maximal 27 Handelspunkte besitzen.
  

	Wenn gegnerische Figuren auf den Feldern mit den Handelssymbolen stehen, so bringen diese keine Handelspunkte ein. 

	Folgenes kann gehandelt werden:
			- Handelspunkte
			- Unverbrauchte Kulturmarker
			- Rohstoffmarker
			- Kulturereigniskarten
			- Unverbindlicher Zusagen

4. ==Bewegungsphase==
	Der Spieler darf jede seiner Einheiten in der Bewegungsphase bewegen. Die Bewegungsweite wird dadurch bestimmt, welche Technologien der Spieler besitzt. (Z.b. Reiterei. (Zu Beginn 2 Felder)
	Die Bewegungen dürfen nicht diagonal geschehen und der Spieler muss eine Figur zuende bewegt haben bevor er die Nächste bewegt.
	Mehrere Figuren eines Spielers können als Gruppe bewegt werden(nicht zwanghaft), wenn sie auf einem Feld stehen (So erhalten sie im Kampf mehr Stärke und können den in der Gruppe befindenen Pionier vor Gegnern schützen.
	Eine Einschrankung für das Gruppieren stellt das Figurenlimit der Zivilisation auf einem Feld da(Zu beginn 2). Das Limit kann durch Technologienerwerb vergrößert werden.
	Figuren können sich nicht über Wasser bewegen, außer der Spieler besitzt eine Technologie, die es erlaubt.
	Unerforschte Spielkartenbereiche können nicht betreten werden, außer sie werden aufgedeckt.
	Pioniere können Felder auf denen Gegner stehen nicht betreten, Krieger allerdings schon.

		==Felder aufdecken==
		Wenn der Spieler ein Feld der Bewegungsreichweite einer Figur für eine Runde opfert, kann er so neue Felder aufdecken. (Die Figur bleibt auf einem Feld stehen und ein Angrenzendes wird aufgedeckt.
		Es kann allerdings sein, dass sich auf dem aufgedeckten Feld ein Barbarendorf oder eine Hütte befindet.
		Die Aufdeckung des Feldes gilt nur für den jeweiligen Spieler. Sprich: Der Spieler sieht, was drauf ist und alle Anderen nicht.
	
		Pioniere können Hütten und Barbarendörfer nicht betreten. Das können nur Krieger-Figuren. 
		Wenn Krieger auf das Feld kommen wird ihre Bewegung dort sofort beenden und sie erkunden nun das Dorf oder die Hütte. 

		In Hütten erhält der Spieler per Zufall einen Rohstoff, mit dem er den Bau von Häusern oder Einheiten bezahlen kann. (Danach wird der Rohstoff entfernt.)
		Beim Betreten von Barbarendörfern erscheinen auf dem Feld 3 Barbaren, die von dem Spieler, der nach dem jetzigen dran kommt gesteuert werden darf.
 		Die Art der Barbaren (Kavallarie etc.) wird zufällig entschieden. Alle Barbaren haben Rank I [Mehr dazu im Abschnitt "Kampf"]. (Im Brettspiel durch das Ziehen von den Kavallerie-, Artillerie- oder Infantriestapel.)
		Wenn der Spieler die Barbaren besiegt hat, bekommt er per Zufall entweder Rohstoffe oder einen Marker für berühmte Persönlichkeiten. [Mehr dazu im Abschnitt Allgemeines]
		Wenn der Spieler gegen die Barbaren verliert, sind seine am Kampfbeteiltigten Krieger eliminiert.
